Interactive Linear Algebra

When I was learning linear algebra in college last year, I found it difficult to understand the principles we covered unless I could imagine an animated diagram of what was going on. In order to make it easier for students after me to learn these principles, I have turned these imagined diagrams into real 3D interactive diagrams using Blender’s “geometry nodes” functionality.

These “diagrams” don’t do a great job covering everything about these concepts, but they are a good supplement to a Linear Algebra course/textbook. I will briefly describe what is going on in each of the diagrams, but in order to understand the concept fully, you will need to do some reading.

Also, since this is a passion project, I’m only covering the concepts I think would look cool as an animated, interactive diagram.

Download the Blender project file here if you want to play with these diagrams yourself. Blender is a free and open source application that runs on Linux, Mac, and Windows. Download that program from blender.org.

You can make changes on the GitHub repository if you would like.

Diagram 1 - Vector Addition

A vector is a direction in space. When you add 2 vectors you get a new vector. It doesn't matter what order you add them, because it is a commutative operation.

Diagram 2 - Vector Subtraction

Vector Subtraction. Why is it that when we do (red minus blue) we get a purple vector that is sometimes LONGER than either of them? Vector subtraction refers to subtracting the each of the x, y, and z coordinates of the blue from the red, piecewise.

(0,0,2) - (3,0,0) = (-3,0,2)

Diagram 3 - SPAN of a Vector

The "SPAN" of a vector is the set of all vectors that result when you multiply it by a scalar. In this example, I am using (0.5, 0.5, 0.5)

4 * (0.5, 0.5, 0.5) = (2,2,2) -2 * (0.5, 0.5, 0.5) = (-1,-1,-1)

therefore: (2,2,2) and (-1,-1,-1) are in the "SPAN" of (0.5, 0.5, 0.5)

Diagram 4 - SPAN of a Set of Vectors

The SPAN of a set of vectors is the set of all vectors that result when you take the sum of all the vectors in the set, scaled by any amount.

a * (vector1) + b * (vector2) = (a_vector_in_span)

Red and blue vectors are vectors in a set.

Green vectors are scalars of either the red or blue vectors.

Purple vector is the sum of scaled vectors in this set.

Purple plane represents the SPAN of this set.

Diagram 5 - Defining Coordinate Systems

Let's take a step back. Why are we able to refer to a location by coordinates such as (2,1,1) ? A coordinate system is defined by a set of "basis vectors". The basis vectors we are all used to are the standard xyz vectors, but any set of vectors can define a coordinate system.

In this diagram we define a 2 dimensional coordinate system within three dimensional space using the red and blue vector as the basis vectors.
